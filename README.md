# Spatial Analysis and Modeling — UniTo 2026

Course material: slide decks and hands-on notebooks.

## Layout

- `slides/` — slide decks (PDF).
- `notebooks/` — Jupyter notebooks + `data/`, `requirements.txt`.

> Editable slide sources (Marp markdown, Keynote files, image assets) are maintained
> separately and are not included in this repository.

## Run the notebooks

```bash
cd notebooks
pip install -r requirements.txt
jupyter lab
```

## Credits

The material adapts open course resources — Michael Szell's *Geospatial Data Science*
(IT University of Copenhagen), Dani Arribas-Bel's *Geographic Data Science*,
Rey/Arribas-Bel/Wolf's *Geographic Data Science with Python*, and the PySAL ecosystem.
